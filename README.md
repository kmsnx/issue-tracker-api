# Issue Tracker Api
[DEMO](https://xmk-dev.github.io/issue-tracker-web/)

Simple issue tracker REST API created in Node.js.

[Checkout issue tracker React app](https://github.com/xmk-dev/issue-tracker-web)

### Development
* Install
    ```
    npm ci --legacy-peer-deps
    ```

* Start
    ```
    npm run start
    ```
